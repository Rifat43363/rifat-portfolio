# Rifat Portfolio Website

Professional portfolio website for **Md Readul Islam Rifat** — MBA in Marketing, BBA in Finance & Banking.

Showcasing expertise in AI-powered automation, Power BI dashboards, business intelligence, and marketing operations. Comprehensive CV integrated with interactive project gallery, skill filters, and comprehensive contact details.

## Files
- `index.html` — main website structure with full CV content
- `styles.css` — full styling with responsive design
- `script.js` — interactions, filters, copy buttons, active nav, back-to-top
- `Md_Readul_Islam_Rifat_CV.pdf` — downloadable CV
- `assets/images/` — profile image and logos used in the site

## How to run
1. Open this folder in VS Code
2. Open `index.html` directly in a browser, or use a local server extension like Live Server
3. Live site: https://rifat43363.github.io/rifat-portfolio/

## Sections
- **Hero** — Professional introduction with quick links
- **Overview** — Capabilities at a glance (6 key areas)
- **About** — Professional narrative and core strengths

- **Education** — MBA (IBA), BBA (Islamic University), HSC & SSC
- **Skills** — Detailed technical, automation, analytics, and business skills
