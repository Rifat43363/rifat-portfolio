## Run locally

1. Extract the ZIP file.
2. Open the folder in VS Code.
3. Open `index.html` directly in a browser, or use the Live Server extension.

## Deploy to GitHub Pages

Replace the existing repository files with the contents of this folder, commit the changes, and push to the branch currently used by GitHub Pages.
