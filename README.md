# Rifat Portfolio Website

This package is ready to open and continue in VS Code.

## Files
- `index.html` — main website structure
- `styles.css` — full styling
- `script.js` — interactions, filters, copy buttons, active nav, back-to-top
- `Md_Readul_Islam_Rifat_CV.pdf` — downloadable CV
- `assets/images/` — profile image and logos used in the site

## How to run
1. Open this folder in VS Code
2. Open `index.html` directly in a browser, or use a local server extension like Live Server

## Main refinements included
- Open Sans typography
- Cleaner visual hierarchy
- Stronger contrasting primary and accent colors
- Improved hover and rollover behavior
- Externalized assets instead of large embedded base64 images
- Interactive logo shortcuts
- Interactive project filters
- Related links between skills, projects, and experience
- Copy-to-clipboard actions for contact details
- Mobile menu improvements
- Scroll progress bar and back-to-top button

## Editing notes
- Change colors in `:root` inside `styles.css`
- Update content in `index.html`
- Add more interactivity in `script.js`
- Replace images inside `assets/images/` if needed
